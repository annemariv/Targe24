Case-vahendid

Mis on Case-vahendid?

CASE (Computer-Aided Software Engineering Tools) vahendid on programmid, mis toetavad tarkvara arendus- ja haldusprotsessi.

Mille jaoks kasutatakse Case-vahendeid?

CASE-tööriistad aitavad rakendada nii töövahendeid kui ka meetodeid kvaliteetsete süsteemide loomiseks. Näiteks CASE-vahendid aitavad automatiseerida tegevusi süsteemiarenduse elutsükli erinevates faasides.

Kuidas jaotatakse Case-vahendeid?

CASE-vahendeid võib jaotada nende otstarbe järgi kahte kategooriasse:

  • ülemise taseme (uppercase)
  • alumise taseme (lowercase)

Milliseid vahendeid saab kasutada prototüüpimise elutsüklis?

Etapid ja näited on järgnevad:

    1. nõuete kogumine - Jira (kliendi või projekti info)

    2. kiire kavandamine - Visual Studio Code (koodi kirjutamine)

    3. iteratsioon - GitHub (haldus)

Milliseid vahendeid olen ise kasutanud?

  • Visutal Studio Code
  • IntelliJ
  • NotePad
  • GitHub
  • Jira (Atlassian)
  • Canva
